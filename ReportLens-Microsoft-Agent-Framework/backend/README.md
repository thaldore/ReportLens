# ReportLens Microsoft Agent Framework - Backend

Bu klasör, projenin ana iş mantığını ve .NET tabanlı ajan orkestrasyonunu (Semantic Kernel) içerir.

## Mimari Prensipler
- **Clean Architecture**: Bağımlılıkların tersine çevrilmesi ve katmanlı yapı.
- **Semantic Kernel**: Microsoft'un resmi AI SDK'sı ile ajan yönetimi.
- **MSSQL Vector Search**: Veri ve vektörlerin aynı veritabanında (SQL Server 2022+) yönetimi.
- **Dependency Injection**: Esnek ve test edilebilir servis yapısı.
