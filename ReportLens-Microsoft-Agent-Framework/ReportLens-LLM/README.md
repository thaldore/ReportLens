# ReportLens - AI & LLM Service (Microservice)

Bu klasör, AI/LLM operasyonlarını merkezi bir mikroservis yapısında toplar.

## Mimari Prensipler
- **Isolation**: AI mantığı ana backend'den izole edilerek ölçeklenebilirlik sağlanır.
- **Local-First**: Ollama ve yerel modeller ile tam veri gizliliği.
- **High Performance**: Vektör arama ve model orkestrasyonu için optimize edilmiş API'ler.
