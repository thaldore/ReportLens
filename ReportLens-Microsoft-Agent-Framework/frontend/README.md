# ReportLens Microsoft Agent Framework - Frontend

Bu klasör, projenin .NET tabanlı kullanıcı arayüzü bileşenlerini içerecektir.

## Mimari Prensipler
- **Modular Monolith**: Bileşen bazlı yapı.
- **Modern UI**: Kullanıcı deneyimi odaklı, hızlı ve temiz bir arayüz.
- **Veri Gizliliği**: Tüm işlemler yerel (local) API'ler üzerinden yürütülür.
